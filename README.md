## What we want this Project to do

This project is for people to look up cards the presumably already have or have heard about:

You will type in the name of the card into a text box on the UI

You will press the button next to the text box in order to search for a card with the name you inputed

If there are no issues, you will be provided with information on the card you searched, such as effects and card type.

(we plan to add more features soon to the description of the card, such as price charts and the works)
